## Description

## Installation

install k6 for load test [instalation](https://k6.io/docs/get-started/installation/)

```bash
$ npm ci
```

create .env file

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
